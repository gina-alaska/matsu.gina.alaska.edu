Matanuska-Susitna Borough's 2011 LiDAR & Imagery Project

Matsu LiDAR and Imagery Services

Web mapping services (WMS) from GINA for the MatSu 2011 collection.

The two primary WMS service end points for the datasets hosted on this mirror are: 

http://matsu.gina.alaska.edu/wms/imagery
http://matsu.gina.alaska.edu/wms/lidar

The WMS currently only supports requests in:

  • project projection: Alaskan State Plane Zone 4 (EPSG:26934)
  • Alaskan Albers (EPSG:3338)
  • geographic (EPSG:4326)

note: everything here is still in development and not feature complete. Let us know if you have problems. Email support@gina.alaska.edu if you have questions
